# Детекция ноутбуков в аудитории

Веб-приложение, которое находит и считает ноутбуки на фотографиях аудиторий. Работает на предобученной модели YOLOv8: пользователь загружает изображение через браузер, сервер прогоняет его через модель, отмечает найденные ноутбуки рамками и показывает их количество. Все обработки сохраняются в историю, которую можно выгрузить в Excel.

Проект выполнен в рамках производственной практики (МТУСИ, 2026).

## Технологии

- Python + Flask — серверная часть и REST API
- YOLOv8 (Ultralytics) — детекция объектов
- OpenCV — чтение изображений и отрисовка рамок
- HTML, CSS, JavaScript — веб-интерфейс
- openpyxl — выгрузка истории в Excel
- JSON — хранение истории обработок

## Запуск

Нужен Python 3.10 или 3.11.

```bash
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux / macOS
pip install -r requirements.txt
python app.py
```

После запуска открыть http://localhost:5000. При первом старте автоматически скачается файл модели `yolov8n.pt` (около 6 МБ).

## Как пользоваться

1. Выбрать изображение и нажать «Запустить обработку».
2. Посмотреть результат: изображение с рамками, количество ноутбуков и время обработки.
3. При необходимости открыть историю обработок и выгрузить её в Excel.

## Endpoints

- `GET /` — главная страница
- `POST /process` — обработка загруженного изображения, ответ в JSON
- `GET /history` — история обработок
- `POST /clear-history` — очистка истории
- `GET /export` — выгрузка истории в Excel
- `GET /health` — проверка состояния

## Как это работает

Изображение читается через OpenCV и передаётся модели YOLOv8n с параметрами `conf=0.5, imgsz=1280`. Модель обучена на наборе COCO, где ноутбук — это класс 63; из всех найденных объектов оставляются только ноутбуки, вокруг них рисуются рамки, после чего они подсчитываются. Модель предобученная, дообучение не выполнялось.
